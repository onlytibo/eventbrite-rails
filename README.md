# README
Ruby 2.5.1
Rails 5.2.3

## Heroku app

`http://eventbrite-rails.herokuapp.com/`

## Install

`git clone https://github.com/thibautgallice/eventbrite-rails.git`

`cd eventbrite-rails/`

`bundle install`

`rails db:create`

`rails db:migrate`

`rails db:reset`

`rails server`

`.env` is needed.

## Run

`Browser > http://localhost:3000/`
